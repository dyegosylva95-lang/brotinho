📋 Passos para subir

01
🗄️
Criar banco no Supabase
⏱ ~15 min · Grátis
  1. Acesse supabase.com → "Start your project"
  2. Crie um projeto: nome brotinho, região South America
  3. Vá em SQL Editor → New Query
  4. Cole o conteúdo de supabase-schema.sql e clique Run
  5. Vá em Settings → API e copie a URL e a chave anon
💡 O schema cria automaticamente o perfil do usuário quando ele se cadastra — você não precisa fazer nada extra!
02
🤖
Pegar chave da Anthropic (IA)
⏱ ~5 min · ~$5 para começar
  1. Acesse console.anthropic.com e crie uma conta
  2. Vá em API Keys → Create Key
  3. Copie a chave — ela começa com sk-ant-
  4. Adicione $5 de crédito em Billing (dura meses!)
03
💻
Configurar e testar localmente
⏱ ~10 min
Precisa ter o Node.js 18+ instalado (nodejs.org). 
cd brotinho
npm install
cp .env.example .env.local
# Preencha o .env.local com suas chaves
npm run dev
Abra http://localhost:3000 e teste! ✅
04
☁️
Deploy na Vercel
⏱ ~10 min · Grátis
  1. Crie conta em vercel.com com GitHub
  2. Suba o código no GitHub: 
    git init && git add .
git commit -m "🌱 Brotinho v1"
git remote add origin https://github.com/SEU_USER/brotinho.git
git push -u origin main
  3. Na Vercel: Add New Project → importar do GitHub
  4. Adicione as 3 variáveis de ambiente:
    NEXT_PUBLIC_SUPABASE_URL NEXT_PUBLIC_SUPABASE_ANON_KEY ANTHROPIC_API_KEY
  5. Clique Deploy — aguarde ~2 minutos
🎉 Pronto! Seu app estará em https://brotinho.vercel.app — acessível de qualquer celular do mundo!
05
🌐
